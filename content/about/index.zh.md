你好，我是 **marsevilspirit**。

一个出生在中国东北，现求学于西安（西邮物联网工程）的软件工程师。 主要使用的编程语言是 Go, C/C++, Rust，我享受着这种向底层不断下潜的过程。

对我而言，**Neovim** 不仅仅是一个编辑器，它是我享受编程的先决条件。我迷恋在终端中完全掌控一切的自由感，那种指尖与逻辑直接连接的体验，是我编程乐趣的源泉。

在代码哲学上，我是一名**极简主义者**。我鄙视过度的封装和无谓的复杂性，始终追求代码的优雅与直观。

**我的旅程**:

- **西邮 Linux 兴趣小组成员:** 很有幸成为西邮 Linux 兴趣小组 2023 级成员，正式加入了这个实验室，才让我走上了学习编程的道路。
- **2025 Google Summer of Code (GSoC):** 在 Apache Dubbo 社区度过了一个充实的夏天，不仅完成了技术的精进，更体会到了开源世界的协作魅力。
- **OceanBase 实习:** 目前我正在探索分布式数据库的深水区。Paxos 协议、存储引擎、分布式事务……在这个庞大的系统中，我正贪婪地吸收着关于“一致性”与“高可用”的一切。

这里是我的**数字自传**。我用它来记录技术上的踩坑与成长，也用它来对抗技术探索中的孤独感。希望这里留下的文字，能成为我与世界沟通的另一种方式。
